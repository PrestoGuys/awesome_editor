About
    The Awesome Editor is a code editor written in Python 
using Tkinter. It is still being developed. It was made by
PrestoGuys in 2024.


Key Commands
1. Exiting
	Exit - Control+Q
2. Cut, Copy, & Paste
	Cut - Control+X
	Copy - Control+C
	Paste - Control+V
